# GEONEX-utils 
If it helps you please give me a star or feedback

Features :
* Generate download list as needed
* Apply data corrections
* Single point extraction tool
* Resampling from time series ( can add the fluxnet data you need )
* ......

Dependencies:
- python 3
- numpy
- pandas
- tqdm
- pyhdf
- glob
- regex
